Guide:
=======

1. Clone this.
2. npm install
3. Copy .env.example to .env
4. gulp serve
5. gulp serve:dist to view the built version.
6. deploy to heroku using git.


Tech Used:
=======

1. WebPack
2. WebPack Hot Reload for React
3. BrowserSync
4. Gulp
5. KeyStone.js
6. React.js
7. React Router
8. React Redux
9. Socket.io
